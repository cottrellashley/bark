Workspace

Use the terminal below to edit your site content. Changes are detected automatically and the page will update live.

Try running claude to use AI-assisted content generation, or use standard shell commands to create and edit markdown files.

Quick Commands

Command What it does
opendoc build /workspace Rebuild the site
ls /workspace/content/ List content files
claude Start Claude Code AI assistant
vim /workspace/content/index.md Edit the homepage
