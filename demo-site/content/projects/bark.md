# Bark

Bark is a static site generator built with Python. It takes markdown content organised into collections and produces a clean, professional static website.

## Features

- **Collections**: Organise content into named groups (writing, projects, notes, etc.)
- **Layouts**: Timeline, grid, or minimal index pages per collection
- **Dark mode**: System-aware with manual toggle
- **Margin notes**: Tufte-style sidenotes with interactive widgets
- **Fast builds**: Pure Python, no JavaScript build step

## Tech Stack

- Python 3.10+
- Jinja2 for templating
- Python-Markdown for content
- Pygments for syntax highlighting
