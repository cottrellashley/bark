# Interactive Console

The OpenDoc TUI console — browse docs, run build commands, and navigate your site from the browser.

<div class="opendoc-terminal" data-mode="tui" data-height="500px"></div>

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `1` | Home screen |
| `2` | Browse Docs |
| `3` | Commands |
| `↑` / `↓` | Navigate items |
| `Enter` | Select / expand |
| `Esc` | Collapse detail |
| `q` | Quit console |
